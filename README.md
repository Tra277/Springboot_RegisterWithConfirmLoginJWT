# Springboot_RegisterWithConfirmLoginJWT

A Restful API for user registration with email confirmation and login with JWT. Inspired by [Code Amigos](https://www.youtube.com/@amigoscode).

## Features
- User Registration
- Email Confirmation
- JWT Authentication
- Secure Login

## Technologies Used
- Spring Boot
- Spring Data JPA
- Spring Security
- Spring Boot Starter Mail
- PostgreSQL
- JSON Web Tokens (JWT)
- Lombok
- IDE: Intelij IDEA 2022 Ultimate
## Getting Started

### Prerequisites
- Java 21
- Maven
- PostgreSQL
